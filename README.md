# Sdmist.github.io

Hello there! 

This repo and site [sdmist.github.io](sdmist.github.io) contains my solutions to the problemset of Codeforces. The solutions are available in multiple languages (currently Python, C and C++). Each solution has an explanation and a Time & Space Complexity Analysis. Each code is original and I have typed each one by hand, thus verifying each one. I have indeed taken help of generative AI to design the site, learn syntax in multiple languages and to optimize the codes or sometimes to find better algorithm, yet no problem solution code has gone without my personal testing. Read [Documentation](sdmist.github.io/documentation.html) for more details.

I hope this can become an interesting solutions archive!
